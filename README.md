# Student Marks Prediction using Machine Learning

## Project Overview

This project predicts student academic performance using Machine Learning. A Linear Regression model is trained on student data such as study hours, attendance percentage, and assignment completion to estimate final marks.

## Features

* Data preprocessing using Pandas
* Data visualization using Matplotlib
* Machine Learning model using Scikit-learn
* Student marks prediction based on input features
* Model accuracy evaluation

## Technologies Used

* Python
* Pandas
* Matplotlib
* Scikit-learn

## Dataset Attributes

* Study Hours
* Attendance Percentage
* Assignments Completed
* Marks Obtained

## Workflow

1. Load and analyze the dataset.
2. Visualize the relationship between study hours and marks.
3. Split data into training and testing sets.
4. Train a Linear Regression model.
5. Evaluate model performance.
6. Predict marks for new student data.

## Sample Output

* Model Accuracy: 99.19%
* Predicted Marks: 76.06

## Learning Outcomes

* Understanding supervised learning concepts
* Working with Linear Regression models
* Data preprocessing and visualization
* Model training and prediction using Scikit-learn
